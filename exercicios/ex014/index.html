CSS Internal

Um estilo interno é aquele que fica configurado dentro da área <head> do seu documento

A grande vantagem dessa maneira em relação a do modelo inline é que podemos escrever uma configuração única para todos os elementos da mesma família. Por exemplo, todos os parágrafos desse documento estão com a mesma configuração de cor e eu só escrevi isso uma vez.

A grande desvantagem dessa técnica em relação ao uso de um CSS Externo (veja o proximo exercício também) é que o estilo não pode ser reaproveitado para outras páginas.

IMPORTANTE: Você pode misturar CSS inline e CSS internal em um mesmo documento sem problemas! Olha só o código fonte dessa página em seu navegador!
